# 🚑 Emergency Accident Prediction System - Frontend

Professional web application with React, Framer Motion, and modern design.

## 🚀 Quick Start

```bash
cd frontend
npm install
npm run dev
```

Visit `http://localhost:5174`

## 📱 Pages

- **Home** `/` - Landing page with features
- **Dashboard** `/dashboard` - Analytics and charts
- **About** `/about` - Team information
- **Contact** `/contact` - Contact form

## 🛠️ Tech Stack

React 19.2 • Vite 7.2 • Framer Motion • Recharts

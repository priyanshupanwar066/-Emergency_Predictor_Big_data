## 🎬 Animation Examples

### 1. Hero Section
```
⏱ Timeline:
0ms  ────────────────────────────► Badge fades in
200ms ────────────────────────────► Title fades in
400ms ────────────────────────────► Subtitle fades in
600ms ────────────────────────────► Buttons fade in
800ms ────────────────────────────► Stats slide in
      (Continuous: Orbs track mouse, shapes float)
```

### 2. Feature Cards
```
⏱ Timeline (Staggered):
0ms   ─► Card 1 slides up
200ms ─► Card 2 slides up
400ms ─► Card 3 slides up
600ms ─► Card 4 slides up
800ms ─► Card 5 slides up
1000ms ─► Card 6 slides up

🎯 On Hover:
- Scale: 1.0 → 1.05
- Shadow: Light → Dark
- Y Position: 0 → -10px
```

### 3. Scroll Animations
```
Before Scroll ──────► In Viewport ──────► After Scroll
(Hidden)           (Visible)           (Persists)
Opacity: 0         Opacity: 1          Opacity: 1
Y: 20px            Y: 0px              Y: 0px
```
